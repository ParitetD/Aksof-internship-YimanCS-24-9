## Выполненные задания

### A1 — Карточка профиля пользователя
Карточка пользователя по центру страницы с аватаром, именем, должностью и тремя кнопками. Использованы семантические теги, box-shadow, border-radius и hover эффект на кнопках.

### A2 — Адаптивная навигация
Навигационная панель с логотипом слева и меню справа. На мобильных устройствах меню скрывается и появляется кнопка гамбургер. Использованы Flexbox, медиа-запросы и JavaScript для открытия/закрытия меню.

### A3 — Каталог товаров
Страница каталога с 8 карточками товаров в сетке. На большом экране 4 колонки, на планшете 2, на телефоне 1. При клике на кнопку "В корзину" текст меняется на "Добавлено ✓". Использован CSS Grid.

### A4 — Анимированная форма входа
Форма входа с анимированным фоном, floating label эффектом и валидацией. Email проверяется на наличие @, пароль на минимум 6 символов. При ошибке поле подсвечивается красным.
